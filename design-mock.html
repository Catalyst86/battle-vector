░░░ DIRECTION / TACTICAL·OPS·TERMINAL ░░░

The existing nomenclature — VEC.BLUE, ARENA, GROUND ZERO, APEX GRID — reads like military call-signs. I doubled down on that. This menu is recast as a combat operations terminal, not an app. Every surface wears L-shaped corner brackets, every number is tabular-aligned with leading zeros, every control leads with a function label and decoration follows.

Typography. Michroma for display — wide, 90° letterforms that read as machined metal, never used in consumer apps. Paired with JetBrains Mono for data readouts (terminal authority, tabular numerals). No body sans. The absence is intentional — this is a HUD, not a brochure.

Color. Existing palette survives but is tightened. Cyan pushed to #39F0FF (electric, instrument-cluster bright). Gold warmer. Background a touch cooler. Red held strictly in reserve for the opponent. White-ish body text shifts from #E5E7EB (sterile grey) to #F2EEE6 (warm bone). That one swap kills most of the flat "AI-generated" feel before any layout changes.

Signature motifs. (1) L-shaped corner brackets everywhere — the UI feels clipped together rather than rounded-soft. (2) Segmented bars for XP and mana, not smooth fills — they read as discrete ammo counts. (3) Tabular numbers with leading zeros everywhere (0607 G, LVL·02). (4) A fine 24px technical-drawing grid under every surface. (5) Chamfered 45° corners on primary buttons, not rounded rectangles. (6) Faint scanlines across the whole display.

Intuitive wins. Upgrades are inline on each card tile (no more hunting for a grey strip — the UPGRADE row has its own gold bar at the bottom of every unlocked card). Role is color-coded and carries a top stripe, so your eye groups by function at a glance. The XP bar segments map to discrete states (reads "halfway through level"). The bottom battle row is weighted 2×1×1 — 1v1 is primary, emphasised and glowing; 2v2 and TUTORIAL clearly secondary. Emoji-free — only BMP glyphs (★ ◆ ▸ → ● ○ ▾) that render reliably on Android's default Godot font.

Screens render at their native 360×780 below. Both phones are static — no scrolling — so you can see the complete rest layout. A tap / hover will subtly surface button states.
